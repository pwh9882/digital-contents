# Insight Mode 콘텐츠 라이브러리

**문서 목적**: Insight Mode에서 사용할 10개 문장 쌍의 완전한 리스트 및 심리학적 근거

---

## 콘텐츠 설계 원칙

### 과학적 근거
- **Self-Perception Theory** (Bem, 1972): 자신의 선택을 관찰하며 자기 이해 형성
- **Choice-Based Assessment**: 강제 선택(forced-choice)은 사회적 바람직성 편향 감소
- **Expressive Writing**: 감정을 언어화하는 과정 자체가 치료적 효과

### 문장 작성 원칙
1. **대조성**: 두 문장은 명확히 반대되는 태도/관점
2. **중립성**: 어느 쪽도 "정답"이 아님
3. **공감성**: 일상에서 흔히 느끼는 감정/생각
4. **간결성**: 15-30자 (타이핑 부담 최소화)
5. **한국 문화 적합성**: 한국 성인의 정서와 가치관 반영

---

## 10개 문장 쌍 전체 리스트

### 카테고리 1: 자기인식 (Self-Perception)

**측정 목표**: 자존감 수준, 자기비판 경향, 외적 검증 의존도

#### 문장 쌍 1 (SP1)

| 요소 | 내용 |
|------|------|
| **문장 A** | 나는 내 장점과 강점을 잘 알고 있다 |
| **문장 B** | 나는 내 단점이 더 많이 눈에 띈다 |
| **측정 차원** | 자존감 / 자기인식의 긍정성 |
| **심리학적 의미** | A 선택: 건강한 자기인식, 강점 기반 사고<br>B 선택: 자기비판 경향, 부정적 자기초점 |
| **글자 수** | A: 19자, B: 17자 |

**프로파일 매핑**:
- A 선택 → 자존감 양호
- B 선택 → 자존감 향상 필요

---

#### 문장 쌍 2 (SP2)

| 요소 | 내용 |
|------|------|
| **문장 A** | 나는 충분히 잘하고 있다고 느낀다 |
| **문장 B** | 항상 더 잘해야 한다는 압박감이 있다 |
| **측정 차원** | 성취 압박 / 자기수용 |
| **심리학적 의미** | A 선택: 자기수용, 현재 만족<br>B 선택: 완벽주의 경향, 성취 압박 |
| **글자 수** | A: 18자, B: 20자 |

**프로파일 매핑**:
- A 선택 → 자기수용 양호
- B 선택 → 완벽주의 경향, 자존감 지원 필요

**관련 연구**:
- Hewitt & Flett (1991): 완벽주의와 우울/불안의 상관관계
- Neff (2003): 자기연민(self-compassion)이 정신건강 증진

---

### 카테고리 2: 스트레스 대응 (Stress Response)

**측정 목표**: 통제감, 스트레스 대처 전략, 낙관성

#### 문장 쌍 3 (SR1)

| 요소 | 내용 |
|------|------|
| **문장 A** | 어려움이 와도 결국 해결할 수 있다고 믿는다 |
| **문장 B** | 문제가 생기면 감당하지 못할 것 같다 |
| **측정 차원** | 통제감 / 자기효능감 |
| **심리학적 의미** | A 선택: 높은 자기효능감, 문제해결 지향<br>B 선택: 낮은 통제감, 무력감 |
| **글자 수** | A: 23자, B: 19자 |

**프로파일 매핑**:
- A 선택 → 스트레스 관리 능력 양호
- B 선택 → 스트레스 관리 지원 필요

**관련 연구**:
- Bandura (1997): 자기효능감이 스트레스 대처에 핵심
- Lazarus & Folkman (1984): 통제 가능성 평가가 스트레스 반응 결정

---

| 요소 | 내용 |
|------|------|
| **문장 A** | 나쁜 날도 있지만 곧 나아질 거라 믿는다 |
| **문장 B** | 힘든 상황이 계속될 것 같아 불안하다 |
| **측정 차원** | 낙관성 / 희망 |
| **심리학적 의미** | A 선택: 낙관적 사고, 일시성 인식<br>B 선택: 비관적 사고, 영속성 믿음 |
| **글자 수** | A: 21자, B: 20자 |

**프로파일 매핑**:
- A 선택 → 낙관성 양호
- B 선택 → 스트레스 관리 지원, 인지 재구성 필요

**관련 연구**:
- Seligman (1990): 설명 양식(explanatory style)과 우울의 관계
- Snyder (2002): 희망 이론 - 목표, 경로, 동기

### 카테고리 3: 사회적 에너지 (Social Energy)

**측정 목표**: 내향성/외향성, 에너지 충전 방식 (치료 개입과 무관, 개인화 용도)

#### 문장 쌍 5 (SE1)

| 요소 | 내용 |
|------|------|
| **문장 A** | 혼자 있는 시간이 나를 회복시킨다 |
| **문장 B** | 사람들과 함께 있을 때 에너지를 얻는다 |
| **측정 차원** | 에너지 원천 / 내향성-외향성 |
| **심리학적 의미** | A 선택: 내향적 경향, 독립적 회복<br>B 선택: 외향적 경향, 사회적 회복 |
| **글자 수** | A: 17자, B: 20자 |

**프로파일 매핑**:
- A 선택 → 내향형 (Therapy 문장 중 조용한 성찰 문구 추천)
- B 선택 → 외향형 (Therapy 문장 중 사회적 지지 문구 추천)

**주의**: 내향/외향은 정신건강과 직접 관련 없음. 개인화 목적으로만 사용.

| **문장 B** | 다양한 사람들과 가볍게 교류하는 걸 즐긴다 |
| **측정 차원** | 관계 선호 / 친밀감 깊이 |
| **심리학적 의미** | A 선택: 깊은 관계 선호, 선택적 친밀감<br>B 선택: 넓은 관계망 선호, 다양성 추구 |
| **글자 수** | A: 22자, B: 22자 |

**프로파일 매핑**:
- 내향/외향 확인용 (SE1과 일치하는지 체크)

## 카테고리별 점수 계산 방법

### 점수 산출 공식

각 카테고리별로:
```
점수 = (A 선택 개수 / 해당 카테고리 문항 수) × 100
```

### 예시 계산

**사용자 선택**:
- SP1: B 선택 (자기인식 - 부정)
- SP2: B 선택 (자기인식 - 부정)
- SR1: A 선택 (스트레스 대응 - 긍정)
- SR2: B 선택 (스트레스 대응 - 부정)
- SE1: A 선택 (사회적 에너지 - 내향)
- SE2: A 선택 (사회적 에너지 - 내향)
- ER1: B 선택 (감정 조절 - 부정)
- ER2: A 선택 (감정 조절 - 긍정)
- FO1: A 선택 (미래 지향성 - 긍정)
- FO2: A 선택 (미래 지향성 - 긍정)

**점수**:
- 자기인식: 0/2 = 0% → 자존감 향상 필요!
- 스트레스 대응: 1/2 = 50% → 경계
- 사회적 에너지: 100% (내향형)
- 감정 조절: 1/2 = 50% → 경계
- 미래 지향성: 2/2 = 100% → 양호

**최종 프로파일**: 자존감 향상형 (가장 낮은 점수)

## 프로파일 분류 알고리즘

### 규칙 기반 분류 (MVP 버전)

```javascript
function assignProfile(scores) {
  // 1. 50% 미만인 카테고리 찾기 (문제 영역)
  const problemAreas = Object.keys(scores)
    .filter(cat => scores[cat] < 50 && cat !== 'social_energy');

  // 2. 가장 낮은 점수 영역 선택
  if (problemAreas.length === 0) {
    return 'balanced';  // 모든 영역 양호
  }

  const lowestCategory = problemAreas
    .reduce((a, b) => scores[a] < scores[b] ? a : b);

  // 3. 프로파일 매핑
  const profileMap = {
    'self_perception': 'self_esteem',       // 자존감 향상형
    'stress_response': 'stress_management', // 스트레스 관리형
    'emotion_regulation': 'emotion_control',// 감정조절형
    'future_orientation': 'motivation'      // 동기부여형
  };

  return profileMap[lowestCategory] || 'balanced';
}
```

### 향후 고도화 가능성 (장기 계획)

- **다중 프로파일**: 여러 영역이 낮을 때 우선순위 설정
- **시간 경과 추적**: 주간/월간 변화 패턴 분석
- **개인화 가중치**: 사용자 피드백 반영
