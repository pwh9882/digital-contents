# Therapy Mode 콘텐츠 라이브러리

**문서 목적**: Therapy Mode에서 사용할 20개 긍정 문구의 완전한 리스트 및 치료적 근거

---

## 콘텐츠 설계 원칙

### 과학적 근거
- **Self-Affirmation Theory** (Steele, 1988): 자기긍정이 스트레스 완충 및 행동 변화 촉진
- **Cognitive Behavioral Therapy (CBT)**: 반복된 긍정적 사고가 부정적 자동사고 교정
- **Spaced Repetition**: 간격을 두고 반복하는 학습이 장기 기억 형성
- **Embodied Cognition**: 신체 행위(타이핑)와 인지(문장 이해)의 결합

### 문장 작성 원칙
1. **실현 가능성**: 믿을 수 있는 수준의 긍정 (과장 금지)
2. **구체성**: 막연한 긍정보다 구체적인 행동/태도 지향
3. **현재형**: "~할 것이다"보다 "~한다" (내면화 촉진)
4. **1인칭**: "나는~" 형태 (자기 진술)
5. **한국 문화 적합성**: 개인주의적 과도함 지양, 균형 잡힌 표현

### 주의사항 (Wood et al., 2009 연구 기반)
- 자존감이 낮은 사람에게 지나치게 높은 긍정문은 역효과
- 예: "나는 모든 면에서 완벽하다" (❌) → "나는 충분히 노력하고 있다" (✅)

**대상**: Insight Mode에서 자기인식 점수 낮음
**목표**: 자기수용, 자기긍정, 내적 가치 인식
**치료적 접근**: Self-Compassion (Neff, 2003), Self-Affirmation (Cohen & Sherman, 2014)

**치료적 접근**: 통제 소재(Locus of Control), 마음챙김, 문제중심 대처

**목표**: 감정 인식, 수용, 건강한 표현
**치료적 접근**: 정서 지능(Emotional Intelligence), DBT 기술, ACT 수용

## 맞춤형 제시 알고리즘

### 세션 구성 (1회 15분 기준)

```javascript
function generateSession(userId) {
  const userProfile = getUserProfile(userId);  // 'self_esteem', 'stress_management', etc.
  const userLevel = getUserTypingLevel(userId);  // 'beginner', 'intermediate', 'advanced'
  const userHistory = getUserSentenceHistory(userId);

  let session = {
    warmup: [],      // 1-2개: 이전 연습 문장
    new: [],         // 3-4개: 새 문장
    challenge: []    // 1개: 도전 과제
  };

  // 1. 워밍업: 이전에 연습한 문장 중 마스터리 < 80%
  const reviewCandidates = userHistory
    .filter(s => s.mastery < 80)
    .sort((a, b) => a.lastPracticedAt - b.lastPracticedAt)  // 오래된 것 우선
    .slice(0, 2);
  session.warmup = reviewCandidates;

  // 2. 새 문장: 프로파일 + 난이도 매칭
  const newCandidates = getSentencesByProfile(userProfile)
    .filter(s => !userHistory.includes(s.id))
    .filter(s => s.difficulty === getDifficultyForLevel(userLevel));
  session.new = shuffleArray(newCandidates).slice(0, 4);

  // 3. 도전 과제: 한 단계 높은 난이도
  const challengeDifficulty = getNextDifficulty(userLevel);
  const challengeCandidates = getSentencesByProfile(userProfile)
    .filter(s => s.difficulty === challengeDifficulty);
  session.challenge = [randomChoice(challengeCandidates)];

  return session;
}

function getDifficultyForLevel(userLevel) {
  const map = {
    'beginner': 'beginner',
    'intermediate': 'intermediate',
    'advanced': 'advanced'
  };
  return map[userLevel];
}

function getNextDifficulty(userLevel) {
  const map = {
    'beginner': 'intermediate',
    'intermediate': 'advanced',
    'advanced': 'advanced'
  };
  return map[userLevel];
}
```

## 마스터리 시스템

### 마스터리 레벨 정의

| 레벨 | 조건 | 의미 |
|------|------|------|
| New | 0회 연습 | 처음 보는 문장 |
| Familiar | 1-2회, 정확도 <95% | 익숙해지는 중 |
| Proficient | 3회+, 정확도 ≥95% | 숙련됨 |
| Mastered | 5회+, 정확도 ≥98% | 완전 내면화 |

### 반복 주기 (Spaced Repetition)

```
1회 연습 후 → 1일 후 복습
2회 연습 후 → 3일 후 복습
3회 연습 후 → 7일 후 복습
4회 연습 후 → 14일 후 복습
5회 연습 → 마스터 (더 이상 강제 제시 안 함, 선택적 복습만)
```
