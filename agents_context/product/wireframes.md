# TheraType UI/UX 와이어프레임

**문서 목적**: 전체 화면별 레이아웃 및 인터랙션 설계

---

## 화면 구조 개요

### 화면 분류

| 분류 | 화면 수 | 화면 목록 |
|------|--------|----------|
| **인증** | 3개 | 랜딩, 회원가입, 로그인 |
| **온보딩** | 2개 | 소개, 기본정보 입력 |
| **Insight Mode** | 4개 | 시작, 문장 선택, 타이핑, 결과 |
| **Therapy Mode** | 4개 | 시작, 문장 제시, 타이핑, 완료 피드백 |
| **대시보드** | 1개 | 진행 추적 |
| **설정** | 2개 | 설정, 프로필 |
| **총계** | **16개** | - |

---

## 화면별 와이어프레임

### 1. 랜딩 페이지 (Landing Page)

```
│         [ 무료로 시작하기 → ]                     │
│                                                  │
│  ┌──────────┐  ┌──────────┐  ┌──────────┐      │
│  │  Insight │  │ Therapy  │  │   진행   │      │
│  │   Mode   │  │   Mode   │  │   추적   │      │
```

```
│  └────────────────────────────────────────────┘ │
│                                                  │
```

**인터랙션**:

- 동의 내역 (timestamp)

**레이아웃**:
```
┌──────────────────────────────────────────────────┐
│  ← 뒤로                                          │
├──────────────────────────────────────────────────┤
│                                                  │
│              당신에 대해 알려주세요                 │
│                                                  │
│  나이대 *                                         │
│  ┌────┐ ┌────┐ ┌────┐ ┌────┐                   │
│  │19-25│ │26-35│ │36-45│ │46+ │                 │
│  └────┘ └────┘ └────┘ └────┘                   │
│                                                  │
│  성별 (선택 사항)                                  │
│  ┌────┐ ┌────┐ ┌────┐ ┌────────┐               │

│  └────┘ └────┘ └────┘ └────────┘               │
│                                                  │
│  타이핑 실력 *                                     │
│  ┌─────────┐ ┌─────────┐ ┌─────────┐           │
│  │  초급    │ │  중급    │ │  고급    │           │
│  │ (<30WPM)│ │(30-50WPM)│ │(50+WPM) │           │
│  └─────────┘ └─────────┘ └─────────┘           │

│  💡 타이핑 실력을 모르시나요?                       │

│                                                  │

│                                                  │

└──────────────────────────────────────────────────┘
```

**인터랙션**:
- 버튼 선택 시 하이라이트 (primary color)
- 필수 항목 미선택 시 "다음" 버튼 비활성화
- "간단 테스트"는 1분간 문장 타이핑 → WPM 측정

**데이터 수집**:
```javascript
{
  userId: "user_123",
  ageGroup: "26-35",
  gender: "female" | "male" | "other" | null,
  typingLevel: "beginner" | "intermediate" | "advanced",
  completedAt: timestamp
}
```

- 키스트로크 모두 로깅

**데이터 수집**:
```javascript
{
```
